方案三

先打质数表再比较，比较复杂但是快一些。

以下段落为 AI 辅助生成，不保证正确性。

const int MAXN = 50000000;
bool is_prime[MAXN + 1];
vector<int> primes;
void sieve() {
    fill(is_prime, is_prime + MAXN + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= MAXN; ++i) {
        if (is_prime[i]) {
            for (int j = i * i; j <= MAXN; j += i) {
                is_prime[j] = false;
            }
        }
    }
    for (int i = 2; i <= MAXN; ++i) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
}

然后用质数表进行分解：

以下段落为 AI 辅助生成，不保证正确性。

for (int prime : primes) {
    if (prime * prime > a) break;
    while (a % prime == 0) {
        a /= prime;
        mp[prime]++;
    }
}
if (a > 1) mp[a]++;

sieve() 生成质数表后，可多次用于分解多个数，适用于需要频繁分解的场景。

公式的展开与推导思路

设 n=pa⋅qb⋅rc⋯n = p^a \cdot q^b \cdot r^c \cdotsn=pa⋅qb⋅rc⋯，其中 p,q,r,…p, q, r, \dotsp,q,r,… 是不同的质数。

1. 单个质数幂的情况：
对于 pap^apa，它的所有因数是：1,p,p2,…,pa1, p, p^2, \dots, p^a1,p,p2,…,pa。
这是一个等比数列，其和为：
σ(pa)=1+p+p2+⋯+pa=pa+1−1p−1\sigma(p^a) = 1 + p + p^2 + \dots + p^a = \frac{p^{a+1} - 1}{p - 1}σ(pa)=1+p+p2+⋯+pa=p−1pa+1−1​
这就是公式中每个因子的来源。

2. 多个质数幂相乘的情况：
当 n=pa⋅qbn = p^a \cdot q^bn=pa⋅qb 时，nnn 的每一个因数都可以唯一地写成 px⋅qyp^x \cdot q^ypx⋅qy 的形式，其中 0≤x≤a0 \le x \le a0≤x≤a， 0≤y≤b0 \le y \le b0≤y≤b。
因此，所有因数的和就等于：
σ(n)=(1+p+p2+⋯+pa)×(1+q+q2+⋯+qb)\sigma(n) = (1 + p + p^2 + \dots + p^a) \times (1 + q + q^2 + \dots + q^b)σ(n)=(1+p+p2+⋯+pa)×(1+q+q2+⋯+qb)
因为第一个括号里的每一项（ppp的幂）都要与第二个括号里的每一项（qqq的幂）相乘，从而生成所有可能的因数。
